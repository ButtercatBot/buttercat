Buttercat is a modular, extensible, and easy to use bot framework for Twitch chat.
It is written in TypeScript and uses the [tmi.js](https://tmijs.com/) library for connecting to Twitch chat.

### Further Reading

- Check out the [examples folder](examples) for working examples

- Check out the [guides](https://buttercat.dev/guides) for more info on what you can do with plugins
